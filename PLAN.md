# Le Filon — Plan de monétisation et roadmap

## Modèle retenu

### Phase 1 : Analyse sectorielle à la demande (maintenant)

- Prix : 349€ par analyse sectorielle complète
- Process : ~30 min de pilotage IA + vérification humaine = ~3€ de coût
- CTA sur le site : "Votre secteur n'y est pas ? Commandez une analyse"
- Posts LinkedIn quotidiens → trafic → commandes
- Objectif : premières ventes cette semaine
- 10K MRR = ~40 analyses/mois

### Phase 2 : Service de validation d'idée (plus tard)

- Prix : 1000-3000€ par mission
- Package : analyse sectorielle + landing page + premiers tests de validation
- Positionnement : "j'ai validé 90 idées, je peux valider la vôtre"
- 10K MRR = 3-10 missions/mois

## Sources de données à intégrer

### Déjà en place

- data.gouv.fr (datasets + APIs + métriques de fréquentation) via MCP
- Recherche web (concurrents, pricing, articles)

### À ajouter (par priorité)

1. **Outil SEO (SEMrush ou Ahrefs)** ~100€/mois — volumes de recherche réels, le maillon faible actuel. Se rembourse avec 1 analyse vendue.
2. **INSEE API** (gratuit) — données économiques, démographie, emploi par territoire. Renforce les TAM/SAM/SOM.
3. **Pappers API** (freemium) — données financières des concurrents (CA, effectifs, date de création). Rend l'analyse concurrentielle vérifiable.
4. **INPI API** (gratuit) — brevets et marques déposées dans le secteur. Détecte l'activité d'innovation.
5. **Google Trends** (gratuit, pas d'API officielle) — tendances de recherche réelles sur le long terme.
6. **EU Open Data** — élargir au-delà de la France pour les secteurs qui s'y prêtent.
7. **ProductHunt / IndieHackers** — valider si quelqu'un a déjà lancé un produit similaire.

### Intégrations envisagées

- MCP INSEE (s'il existe ou à créer)
- MCP Pappers ou appels API directs
- SEMrush API pour automatiser les volumes de recherche dans les fiches

## Distribution / Acquisition

- 1 post/jour tiré des fiches produit
- 90 fiches × 3 angles = 270 posts = 9 mois de contenu
- Chaque post finit par un CTA vers le site ou vers le service d'analyse
- Outil : Buffer ou Typefully pour scheduler (15€/mois)
- À valider : est-ce que LinkedIn convertit avant d'automatiser complètement

- Posts dans r/EntreprendreenFrance (validé : 1183 vues, 14 upvotes)

- Mots-clés cibles : "idée SaaS France", "opportunité open data", "data.gouv.fr business"

- Bannière flottante après 15s
- Contenu : nouvelles analyses hebdomadaires
- Objectif : construire une base email pour les futures offres
